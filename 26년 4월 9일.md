\[하드웨어 종류]



1. CPU : 중앙 처리 장치 (전자 계산기) → 코어

2. RAM : 주 기억 장치 (e.g. 책상 위) → 휘발성

3. HDD \& SSD \& M.2 : 보조 기억 장치 (e.g. 책상 서랍) → 비휘발성

4. GPU : 그래픽 처리 장치 → 3D 연산용

5. 랜카드 : 네트워크 통신 장치

6. 파워서플라이 : 컴퓨터의 심장 (무거울수록 성능 ↑)

7. 메인보드 : 각종 부품을 모아서 처리하는 보드 (마더보드)



\[OSI 7계층] : 호환 문제를 보완하기 위해 통신의 계층을 나누어 처리



1. 물리 계층 → 랜카드, 케이블

2. 데이터 링크 계층 → 스위치 허브

3. 네트워크 계층 : 서로 다른 망을 연결 → 라우터

4. 전송 계층 → TCP (정확성), UDP (속도 향상)

5. 세션 계층 : 통신 유지

6. 표현 계층(presentation) : 암호화, 복호화, utf-8

7. 응용 계층 : 개발용
